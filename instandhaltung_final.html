⚙️ Einrichtung

Verbinde diese App mit deinem Supabase-Projekt. Das dauert ~5 Minuten.

Schritt 1 – Supabase-Konto:
Gehe auf supabase.com → "Start your project" → kostenlos registrieren.

Schritt 2 – Neues Projekt erstellen:
Name: z.B. instandhaltung → Region: Central EU (Frankfurt) → Passwort merken.

Schritt 3 – API-Daten holen:
Im Dashboard: Settings → API → kopiere Project URL und anon public key.
🗄️ Datenbank

Erstelle die nötigen Tabellen in deiner Supabase-Datenbank.

Im Supabase-Dashboard:
Gehe zu SQL Editor → New query → füge den SQL-Code ein → klicke Run.

Kopiere den SQL-Code über den Button unten in die Zwischenablage.
Nach dem SQL-Run: Gehe zu Authentication → Providers → stelle sicher, dass Email aktiviert ist (ist Standard). Dann Authentication → URL Configuration → füge unter "Redirect URLs" hinzu: * (für lokalen Test).
👤 Admin-Konto

Erstelle deinen Administrator-Account.
